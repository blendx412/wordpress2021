== Change log ==
= 1.5.0 (20/08/2019) =
* Fixed: 'Upsell by date' reports by setting correct end date.
* Fixed: WC Compatibility version updated.

= 1.4.0 (25/06/2019) =
* Added: Added additional report of "report by offer". it will show the offer purchase reports on offer level against a funnel.
* Improved: Timeline UI highlighting in case of errors.
* Fixed: Translation text domain corrected.

= 1.3.0 (20/12/2018) =
* Added: New event support "offer refunded".
* Improved: Removed cache from metabox database queries to show the most recent event logs.
* Fixed: Debug file for license check is not removed.

= 1.2.0 (11/09/2018) =
* Added: New event added for the upstroke time-line as "offer skipped". This event gets recorded when upstroke decides to skip the offer when certain validations doesn't match.

= 1.1.0 (27/07/2018) =
* Improved: One function name renamed with local prefix.


= 1.0.0 (24/07/2018) =
* Public Release
